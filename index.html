Exercise: Create a data-cleaning.md Using Claude

In this exercise you will use the free Claude website (claude.ai) to generate a reusable data-cleaning.md instruction file. You will prompt Claude to write the file for you, then download and save it.

💡 Requirement: A free Claude account at claude.ai. No paid plan needed.
📋

Objective

Use the Claude website to generate a Markdown file called data-cleaning.md that serves as a reusable instruction set for cleaning media monitoring datasets. The file should cover:

  • Removing duplicate rows based on URL or headline
  • Filtering out rows containing unwanted keywords
  • Enriching date columns (extract Month, Day, Year)
  • Outputting a cleaned file ready for analysis
🛠

Step-by-Step Instructions

1

Go to the Claude website

Open your browser and navigate to claude.ai. Log in with your free account (or create one).

2

Start a new conversation

Click "New chat" to begin a fresh conversation with Claude.

3

Enter the following prompt

Copy and paste this prompt into the chat box:

📋 Prompt to Copy
Create a Markdown file called data-cleaning.md that serves as a
reusable instruction guide for cleaning media monitoring CSV/Excel
datasets. The file should include:

1. **Purpose** — Explain that this is for cleaning raw media
   monitoring data

2. **Input Requirements** — Describe expected columns:
   Date, Headline, URL, Publication, Content

3. **Cleaning Steps:**
   - Duplicate Removal: deduplicate by URL (fallback to Headline)
   - Keyword Filtering: remove rows with user-specified keywords
     (case-insensitive)
   - Date Enrichment: parse the Date column and add Month, Day,
     Year columns

4. **Output** — Save as a new file with _cleaned suffix,
   include a summary of changes

5. **Rules** — Never modify the original file, preserve
   column order, handle missing Date column gracefully

Format it as a clean, well-structured Markdown document that
someone could follow as step-by-step instructions. Output the
full file content inside a code block.
4

Review Claude's output

Claude will generate the full Markdown content. Review it and ask follow-up questions if needed, for example:

  • "Add a section for handling encoding issues in CSV files"
  • "Include an example of the expected output summary format"
  • "Make the keyword filtering section more detailed"
5

Save the file

Once you are happy with the output:

  1. Click the Copy button on the code block
  2. Open any text editor (Notepad, VS Code, etc.)
  3. Paste the content and save as data-cleaning.md
6

Test it

Start a new Claude chat, upload a sample CSV, and paste the contents of your data-cleaning.md as instructions. Verify that Claude follows the cleaning steps correctly.

✨

Expected Output (Reference)

Your generated data-cleaning.md should look similar to this:

📄 data-cleaning.md
# Data Cleaning Instructions

## Purpose
Clean raw media monitoring datasets by removing duplicates,
filtering unwanted keyword rows, and enriching date columns.

## Input Requirements
- File format: CSV or Excel (.xlsx)
- Expected columns: Date, Headline, URL, Publication, Content
- Additional columns are preserved as-is

## Cleaning Steps

### 1. Duplicate Removal
- Primary key: **URL** column
- Fallback: **Headline** column (if URL is missing)
- Keep the first occurrence; remove subsequent duplicates
- Report: "Removed X duplicate rows"

### 2. Keyword Filtering
- Ask the user for a list of keywords to exclude
- Check Headline and Content columns (case-insensitive)
- Remove any row that contains an excluded keyword
- Report: "Filtered X rows matching excluded keywords"

### 3. Date Enrichment
- Auto-detect the date column (Date, Published, Pub Date)
- Parse into YYYY-MM-DD format
- Add new columns at the end: Month, Day, Year
- Supported formats: MM/DD/YYYY, DD-Mon-YYYY, ISO 8601

## Output
- Save as: [original_filename]_cleaned.[ext]
- Print summary: original rows, duplicates removed,
  rows filtered, final row count

## Rules
- Never modify the original file
- Preserve all original columns in their original order
- New columns (Month, Day, Year) go at the end
- If no Date column exists, skip enrichment and notify user
💬

Tips for Better Results

  • Be specific — the more detail you give Claude, the better the output
  • Iterate — ask Claude to revise sections that don't match your needs
  • Add examples — ask Claude to include sample input/output for each step
  • Test with real data — upload an actual CSV in a follow-up chat to verify the instructions work
  • Version your file — save improved versions as data-cleaning-v2.md, etc.

const DS_CONFIG1 = {
    DAILY_REQUEST_LIMIT: 500,
    API_ENDPOINT: 'https://api.deepseek.com/v1/chat/completions',
    MODEL: 'deepseek-chat',
    MAX_TOKENS: 4096,
    TEMPERATURE: 0.0,
    REQUEST_RETRIES: 2,
    REQUEST_TIMEOUT_MS: 30000,
    PROMPT_DOC_IDS: {
        sentiment: '1JP6WecOtrxfHRbCZK6ROacEFRQaIjfPGENfBUb8IM1M',
        summary: '1kXMZSSrPNx0coqpzqtkzVmGdCfpzJlbUpzfQfR9Hp7M',
        category: '15c5Xu3wEerypXUGgfclGAodAZg8pJdhyieN1TS9iS5Q'
    }
};

function setupCredentials(apiKey) {
    const key = (apiKey || '').toString().trim();
    if (!key) {
        throw new Error('Please provide an API key. Example: setupCredentials("sk-...")');
    }
    PropertiesService.getScriptProperties().setProperty('DS_API_KEY', key);
    return 'API key saved successfully.';
}

function setApiKeyFromPrompt() {
    const ui = SpreadsheetApp.getUi();
    const response = ui.prompt(
        'Set DeepSeek API Key',
        'Paste your API key (it will be saved in Script Properties):',
        ui.ButtonSet.OK_CANCEL
    );
    if (response.getSelectedButton() !== ui.Button.OK) {
        ui.alert('Cancelled.');
        return;
    }
    const result = setupCredentials(response.getResponseText());
    ui.alert(result);
}

function clearApiKey() {
    PropertiesService.getScriptProperties().deleteProperty('DS_API_KEY');
    return 'API key cleared.';
}

function getApiKey_() {
    const apiKey = PropertiesService.getScriptProperties().getProperty('DS_API_KEY');
    if (!apiKey || !apiKey.trim()) {
        return {
            ok: false,
            error: 'Error: API key is missing. Use Prompt Tools > Set API Key.'
        };
    }
    return { ok: true, value: apiKey.trim() };
}

function setupPromptDoc(docIdOrUrl, taskName) {
    const cleanTaskName = normalizeTaskName(taskName || 'sentiment');
    const configuredDocId = (DS_CONFIG1.PROMPT_DOC_IDS &&
        DS_CONFIG1.PROMPT_DOC_IDS[cleanTaskName]) || '';
    const rawInput = (docIdOrUrl !== undefined && docIdOrUrl !== null &&
        docIdOrUrl.toString().trim() !== '')
        ? docIdOrUrl.toString().trim()
        : configuredDocId.toString().trim();
    if (!rawInput) {
        throw new Error('Please provide a valid Google Doc ID/URL.');
    }
    let cleanDocId = rawInput;
    if (rawInput.indexOf('docs.google.com/document') !== -1) {
        const match = rawInput.match(/\/document\/d\/([a-zA-Z0-9_-]+)/);
        if (!match || !match[1]) {
            throw new Error('Could not extract a valid Doc ID from the URL.');
        }
        cleanDocId = match[1];
    }
    const scriptProps = PropertiesService.getScriptProperties();
    scriptProps.setProperty(getPromptDocIdKey(cleanTaskName), cleanDocId);
    return syncPromptFromDoc(cleanTaskName);
}

function syncPromptFromDoc(taskName) {
    try {
        const cleanTaskName = normalizeTaskName(taskName || 'sentiment');
        const scriptProps = PropertiesService.getScriptProperties();
        const configuredDocId = (DS_CONFIG1.PROMPT_DOC_IDS &&
            DS_CONFIG1.PROMPT_DOC_IDS[cleanTaskName]) || '';
        const propertyDocId = scriptProps.getProperty(
            getPromptDocIdKey(cleanTaskName));
        const docId = configuredDocId || propertyDocId;
        if (!docId) {
            return 'Error: Prompt Doc is not configured for "' + cleanTaskName + '".';
        }
        const docText = DocumentApp.openById(docId).getBody().getText().trim();
        if (!docText) {
            return 'Error: Prompt Doc for "' + cleanTaskName + '" is empty.';
        }
        scriptProps.setProperty(getPromptTextKey(cleanTaskName), docText);
        return 'Prompt synced successfully for "' + cleanTaskName + '".';
    } catch (error) {
        return 'Error: Unable to read Prompt Doc - ' + error.toString();
    }
}

function syncAllPromptsFromDocs() {
    const tasks = ['sentiment', 'summary', 'category'];
    const results = tasks.map(function(task) {
        return syncPromptFromDoc(task);
    });
    return results.join('\n');
}

function getCachedSystemPrompt(taskName) {
    const cleanTaskName = normalizeTaskName(taskName || 'sentiment');
    const promptText = PropertiesService.getScriptProperties()
        .getProperty(getPromptTextKey(cleanTaskName));
    if (!promptText || promptText.trim() === '') {
        return 'Error: Prompt for "' + cleanTaskName + '" is not synced yet.';
    }
    return promptText;
}

function normalizeTaskName(taskName) {
    const validTasks = ['sentiment', 'summary', 'category'];
    const normalized = String(taskName || '').trim().toLowerCase();
    if (validTasks.indexOf(normalized) === -1) {
        throw new Error('Invalid taskName. Use one of: sentiment, summary, category.');
    }
    return normalized;
}

function getPromptTextKey(taskName) {
    return 'PROMPT_TEXT_' + taskName.toUpperCase();
}

function getPromptDocIdKey(taskName) {
    return 'PROMPT_DOC_ID_' + taskName.toUpperCase();
}

function parseJsonSafe_(text) {
    try { return JSON.parse(text); }
    catch (error) { return null; }
}

function callDeepSeek_(payload, apiKey) {
    const options = {
        method: 'post',
        contentType: 'application/json',
        headers: { 'Authorization': 'Bearer ' + apiKey,
                    'Content-Type': 'application/json' },
        payload: JSON.stringify(payload),
        muteHttpExceptions: true,
        timeout: DS_CONFIG1.REQUEST_TIMEOUT_MS
    };
    let attempt = 0;
    const maxAttempts = DS_CONFIG1.REQUEST_RETRIES + 1;
    while (attempt < maxAttempts) {
        attempt++;
        try {
            const response = UrlFetchApp.fetch(DS_CONFIG1.API_ENDPOINT, options);
            const status = response.getResponseCode();
            const bodyText = response.getContentText() || '';
            const bodyJson = parseJsonSafe_(bodyText);
            if (status === 200) {
                return { ok: true, data: bodyJson || {} };
            }
            const retryable = status === 429 || status >= 500;
            if (retryable && attempt < maxAttempts) {
                Utilities.sleep(1000 * attempt);
                continue;
            }
            const apiMessage = bodyJson && bodyJson.error && bodyJson.error.message
                ? bodyJson.error.message
                : ('API request failed with status ' + status);
            return { ok: false, error: 'Error: ' + apiMessage, status: status };
        } catch (error) {
            if (attempt < maxAttempts) {
                Utilities.sleep(1000 * attempt);
                continue;
            }
            return { ok: false, error: 'Error: Network/API request failed - ' +
                error.toString() };
        }
    }
    return { ok: false, error: 'Error: API request failed after retries.' };
}

function checkRateLimit() {
    const cache = CacheService.getScriptCache();
    const today = new Date().toDateString();
    const requestCount = cache.get(today);
    if (!requestCount) {
        cache.put(today, '1', 24 * 60 * 60);
        return true;
    }
    const count = parseInt(requestCount);
    if (count >= DS_CONFIG1.DAILY_REQUEST_LIMIT) {
        return false;
    }
    cache.put(today, (count + 1).toString(), 24 * 60 * 60);
    return true;
}

function runAiTask(taskName, text_to_analyze) {
    if (!text_to_analyze || text_to_analyze.toString().trim() === '') {
        return "Cell is empty";
    }
    if (!checkRateLimit()) {
        return 'Error: Daily request limit (' +
            DS_CONFIG1.DAILY_REQUEST_LIMIT + ') exceeded.';
    }
    const apiKeyResult = getApiKey_();
    if (!apiKeyResult.ok) return apiKeyResult.error;
    const systemPrompt = getCachedSystemPrompt(taskName);
    if (systemPrompt.indexOf('Error:') === 0) return systemPrompt;
    const payload = {
        model: DS_CONFIG1.MODEL,
        messages: [
            { "role": "system", "content": systemPrompt },
            { "role": "user",   "content": text_to_analyze }
        ],
        max_tokens: DS_CONFIG1.MAX_TOKENS,
        temperature: DS_CONFIG1.TEMPERATURE
    };
    const result = callDeepSeek_(payload, apiKeyResult.value);
    if (!result.ok) return result.error;
    const content = result.data &&
        result.data.choices &&
        result.data.choices[0] &&
        result.data.choices[0].message &&
        result.data.choices[0].message.content;
    if (!content) return 'Error: API returned an unexpected response format.';
    return String(content).trim();
}

/** @customfunction */
function sentimentTagger(text_to_analyze) {
    return runAiTask('sentiment', text_to_analyze);
}

/** @customfunction */
function summarizer(text_to_analyze) {
    return runAiTask('summary', text_to_analyze);
}

/** @customfunction */
function categoryTagger(text_to_analyze) {
    return runAiTask('category', text_to_analyze);
}

❄ FreezeFunc.gs (copy this entire block)
const AI_CUSTOM_FUNCTIONS = [
  'sentimentTagger',
  'summarizer',
  'categoryTagger'
];

const AI_FORMULA_REGEX = new RegExp(
  '=\\s*(' + AI_CUSTOM_FUNCTIONS.join('|') + ')\\s*\\(','i'
);

/** Freeze AI formulas in the ACTIVE sheet only. */
function freezeApiFormulasActiveSheet() {
  const sheet = SpreadsheetApp.getActiveSheet();
  const range = sheet.getDataRange();
  const formulas = range.getFormulas();
  const values   = range.getValues();
  let frozen = 0;

  for (let r = 0; r < formulas.length; r++) {
    for (let c = 0; c < formulas[r].length; c++) {
      const f = formulas[r][c];
      if (f && AI_FORMULA_REGEX.test(f)) {
        frozen++;
      } else if (f) {
        values[r][c] = f;
      }
    }
  }

  if (frozen > 0) {
    range.setValues(values);
  }

  SpreadsheetApp.getActive().toast(
    frozen + ' AI formula(s) frozen in "' + sheet.getName() + '"',
    'Freeze AI Formulas',
    5
  );
}

Part C — Customize the Configuration

Now you need to modify three things in the DS_CONFIG1 object at the top of Code.gs to make it your own:

Task 1: Change the Daily Request Limit

Find this line near the top of the script:

DAILY_REQUEST_LIMIT: 500,

Change it to a value that fits your usage. For example, to limit to 100 requests per day:

DAILY_REQUEST_LIMIT: 100,
⚠ Why change this? Each API call costs tokens. A lower limit prevents accidental overuse during testing. You can always increase it later.

Task 2: Replace the Google Doc IDs with Your Own

Find the PROMPT_DOC_IDS section:

PROMPT_DOC_IDS: {
    sentiment: '1JP6WecOtrxfHRbCZK6ROacEFRQaIjfPGENfBUb8IM1M',

  1. Open the Google Doc in your browser

  3. Copy the long string between /d/ and /edit

PROMPT_DOC_IDS: {

    summary:   'YOUR_SUMMARY_DOC_ID',
    category:  'YOUR_CATEGORY_DOC_ID'

💡 What are these docs? Each Google Doc contains the "system prompt" that tells the AI how to analyze text. For example, the sentiment doc might say: "Classify the following text as Positive, Negative, or Neutral."

Exercise: Create Your AI System Prompts

In this exercise, you will write three system prompts that power the AI custom functions in Google Sheets. Each prompt will be saved in its own Google Doc and linked to the script via PROMPT_DOC_IDS.

💡 Structure: Every prompt follows the same 4-part format — Role, Instruction, Constraints, Sample Output.
📐

Prompt Structure

All three prompts must follow this consistent format:

SectionPurposeWhat to Write
Role Defines who the AI is A short persona statement. Sets expertise, tone, and perspective.
Instruction Defines what the AI must do The core task. Be specific about input, action, and expected output format.
Constraints Defines boundaries Rules, limits, things to avoid. Keeps the AI focused and consistent.
Sample Output Defines what good looks like A concrete example of the expected response. Anchors the AI's format.
📝

Prompt 1 — Summary Prompt

This prompt powers the =summarizer() function. It takes article text and produces a concise summary.

Instructions

1

Create a new Google Doc and name it "Summary Prompt"

2

Write your prompt following the 4-part structure. Use the reference below as a starting point, then customize it to fit your clients' needs.

Reference Template

📄 Summary Prompt — Google Doc
ROLE:
You are a senior media analyst at a PR monitoring agency.
You specialize in extracting key information from news
articles and producing concise, executive-ready summaries
for client reports.

INSTRUCTION:
Summarize the provided news article text. Your summary must:
- Capture the main story in 1-2 sentences
- Identify WHO is involved (people, organizations)
- State WHAT happened or was announced
- Note WHY it matters or its significance
- Include any relevant numbers, dates, or data points

Output a single paragraph summary of 40-60 words.
Do not use bullet points in the output.

CONSTRAINTS:
- Maximum 60 words; never exceed this limit
- Write in third person, past tense
- Do not include your own opinions or analysis
- Do not add information not found in the original text
- If the text is too short or unclear, respond with:
  "Insufficient content for summary"
- Do not start with "The article discusses..." or
  "This article is about..."
- Use neutral, factual language throughout

SAMPLE OUTPUT:

Input: "Tech giant Acme Corp announced a $50M expansion
of its Manila operations on March 10, creating 2,000 new
jobs in AI development. CEO Maria Santos said the move
positions the Philippines as a key hub for the company's
Asia-Pacific growth strategy."

Output: "Acme Corp announced a $50 million expansion of
its Manila operations on March 10, projected to generate
2,000 new AI development positions. CEO Maria Santos
emphasized the investment strengthens the Philippines'
role in the company's Asia-Pacific growth strategy."

INSTRUCTION:
Analyze the sentiment of the provided news article text

- POSITIVE: Coverage that favorably portrays the subject
  (achievements, awards, growth, praise, endorsements,
  successful initiatives)
- NEGATIVE: Coverage that unfavorably portrays the subject
  (scandals, failures, criticism, losses, controversies,
  legal issues)
- NEUTRAL: Factual reporting with no clear positive or
  negative framing (announcements, routine coverage,
  industry data, event listings)
- BALANCED: Coverage that contains BOTH clearly positive
  AND clearly negative elements in roughly equal measure

Output ONLY the sentiment label. Nothing else.

CONSTRAINTS:
- Respond with exactly ONE word: POSITIVE, NEGATIVE,
  NEUTRAL, or BALANCED
- Do not include explanations, justifications, or analysis
- Do not use quotation marks in your response
- Judge sentiment toward the PRIMARY subject, not
  secondary mentions
- Press releases and routine announcements are NEUTRAL
  unless they contain evaluative language
- Opinion pieces and editorials should be evaluated by
  their stance, not the topic
- If the text is empty, unreadable, or not a news article,
  respond with: NEUTRAL

Slide 7 FreezeFunc

Freeze AI Formulas

The FreezeFunc script solves a critical problem: every time the spreadsheet recalculates, AI custom functions re-run and consume API tokens.

What It Does

  • Scans the active sheet for cells containing sentimentTagger(), summarizer(), or categoryTagger()
  • Replaces those formulas with their current evaluated text values
  • Preserves all other formulas untouched
  • Performs a single batched write for efficiency

How to Use

Go to Prompt Tools > Output > Freeze AI Formulas (Active Sheet)

⚠ Irreversible: Once frozen, the original formulas are replaced. Make sure results are correct before freezing.
